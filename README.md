# Lead Intelligence Manager

Aplicación Java para validar, puntuar, clasificar y hacer seguimiento de leads comerciales.

## Objetivo
El proyecto busca resolver una necesidad real de pequeñas empresas o agencias que gestionan leads de forma desordenada, permitiendo registrar, consultar, filtrar y priorizar contactos comerciales.

## Tecnologías previstas
- Java
- JDBC
- Base de datos relacional
- SQL
- XML / XSD
- GitHub

## Estructura del repositorio
- `docs/bbdd`: documentación de bases de datos
- `docs/sistemas`: informe técnico del entorno
- `docs/xml`: documentación del XML/XSD
- `docs/mpo`: decisiones de arquitectura y mejora estructural
- `docs/entornos`: notas del repositorio y organización
- `sql`: scripts SQL
- `xml`: archivos XML y XSD
- `src`: código fuente Java
- `assets`: capturas y recursos visuales
